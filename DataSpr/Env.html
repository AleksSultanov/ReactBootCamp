Создания окружения

на Win

python -m venv gps_venv
активация \gps_env\Scripts\activate.bat
деактивация \gps_env\Scripts\deactivate.bat

на Ubuntu

установка sudo apt install -y python3-venv
создание окружения python3 -m venv my_env_project
активация source my_env_project/bin/activate
просмотр пакетов pip list
список в файл пакетов pip freeze > requirements.txt
установка из файла pip install -r requirements.txt --no-cache-dir
