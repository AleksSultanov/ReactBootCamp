Шпаргалка по flask

Запуск в коде 

  if __name__ == "__main__":
      app.run()

	  
     app.run(host='0.0.0.0') #-  приложение начнет прослушивать все общедоступные IPs.Запуск с локальным ip
     app.run(host='0.0.0.0', port=4000) #- можете указать любой конкретный порт, на котором будет работать ваше приложение, скажем 4000 (порт по умолчанию-5000)

Запуск из коммандной строки 

 
  flask run 
  flask run --host=0.0.0.0

Включить режим debug 

 
  # ubuntu
  export FLASK_ENV=development
  # win
  set FLASK_ENV=development
  # win powershell
  $env:FLASK_ENV = "development"
