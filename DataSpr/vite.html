Установка vite

        yarn add vite

Создание нового проекта React

        yarn create vite my-react-app --template react
          (где my-react-app это имя проекта)
        yarn
          (установка пакетов)
        yarn add gh-pages --save-dev  
           (для деплоя на git)
            в файле package.json
                добавляем где name и private
                "homepage": "https://alekssultanov.github.io/<название проекта на git>",
                добавляем в scripts
                "predeploy": "npm run build",
                "deploy": "gh-pages -d dist",  
                перед "build"
            в файле vite.config.js 
            base: "/<название проекта на git>/",


        yarn dev
          (Запуск)

Установка проекта по готовым файлам/p>
