Шпаргалка по git

установка 

 
   sudo apt install git-all

Задать начальный конфиг

 
  git config --global user.name "Aleks Sultanov"
  git config --global user.email aesultanov@yandex.ru



 Добавление и выгрузка


 
  создать репозитарий
  git init  
  
  Добавить файлы из тек.каталога
  git add .
  
  узнать статус 
  git status
  
  фиксация
  git commit -m "Initial commit"
  
  выгрузка в хранилище
  git remote add origin https://gitlab.com/AleksSultanov/lessonflask.git
  git push --set-upstream origin master
  надо перейти по ссылке и подтвердить
  
  
  выгрузить и переключить ветку
  git clone https://gitlab.com/AleksSultanov/userapy.git
  git checkout master

  создание бранча
  git branch <название ветки>
  переключить 
  git checkout <название ветки>
